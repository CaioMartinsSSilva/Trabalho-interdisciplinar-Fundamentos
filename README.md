# Hotel Descanso Garantido — Versão Resumida (Funcionalidades 1–4)

Trabalho interdisciplinar (FES + AED1).  
Esta versão implementa apenas as funções obrigatórias para alunos de FES:

1. Cadastro de Cliente  
2. Cadastro de Funcionário  
3. Cadastro de Quarto  
4. Cadastro de Estadia (com validação de datas e disponibilidade)

Inclui:
- Código em C
- Documentação das funções
- Backlog + Sprints
- Testes planejados
- Relatório de testes (modelo)

## Como compilar
gcc -o hotel src/*.c

## Como executar
./hotel
